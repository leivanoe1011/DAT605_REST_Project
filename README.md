```
index.html
package.json
node_modules/
css

├── controllers/

```

```
$ npm install --save todomvc-common
```
* css files are [here](https://github.com/tastejs/todomvc-app-css)
```
$ npm install --save todomvc-app-css
```
* for data persistence, use RESTful API to send JSON for database queries
